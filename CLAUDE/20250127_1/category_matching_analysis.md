# カテゴリマッチング精度改善 - 調査分析レポート

## 1. 現状の問題点

### 1.1 致命的なバグ: 正規表現の不備

**match_categories.py:121行目** の正規表現に問題がある。

```python
# 現在のコード
t = re.sub(r"[^0-9a-zA-Zぁ-んァ-ン一-龥\s]+", " ", t)
```

**問題**: カタカナ長音記号「ー」(U+30FC) が許可範囲外

| 文字 | Unicode | 範囲内か |
|------|---------|----------|
| ァ | U+30A1 | Yes |
| ン | U+30F3 | Yes |
| ー | U+30FC | **No** |
| ヴ | U+30F4 | No |

**結果**:
- 「ギター」→「ギタ」
- 「アコースティック」→「アコ スティック」
- 「パーカッション」→「パ カッション」

これにより、synonyms.tsvに `guitar → ギター` の対応があっても、「ギタ」にはマッチしない。

### 1.2 synonyms.tsvの状況

| 項目 | 値 |
|------|-----|
| 総行数 | 7,740行 |
| 日本語対応あり | 7,361件 (95.1%) |
| 日本語対応なし | 379件 (4.9%) |

**辞書は存在するが、正規表現バグにより機能していない。**

### 1.3 第2階層マッピングの欠如

現在は `level_1_cat.txt` による第1階層マッピングのみ。
第2階層のマッピングがないため、以下のような誤マッチが発生：

| Amazon カテゴリ | 現在のマッチ先 | 正しいマッチ先 |
|---------------|--------------|--------------|
| Hunting & Fishing/Archery | フィッシング | 弓道 |
| Sports/Airsoft | ゴルフ | サバイバルゲーム |
| Drums & Percussion | 弦楽器 | ドラム・パーカッション |

### 1.4 構造的な不一致

**Amazon第2階層の例（Sports & Outdoors）**:
- Sports (952件) - 大括り
- Outdoor Recreation (605件)
- Hunting & Fishing (343件) - 複合カテゴリ

**Wowma第2階層の例（スポーツ・アウトドア）**:
- 野球 (90件) - 個別スポーツ
- ゴルフ (76件)
- フィッシング (100件)
- 弓道 (7件)

→ 粒度が異なるため、単純なfuzzyマッチでは対応困難

## 4. 根本原因の整理

### 優先度: 高

1. **正規表現バグ** - 長音記号が削除される
2. **第2階層マッピングなし** - 誤マッチの主要因

### 優先度: 中

3. **synonyms.tsv不足** - 特殊なカテゴリ名の翻訳不足
4. **マッチングロジック** - 階層構造を考慮していない

### 優先度: 低

5. **閾値調整** - auto_threshold 0.7 は高すぎる可能性
